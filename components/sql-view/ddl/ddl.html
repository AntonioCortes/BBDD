Data Definition Language (DDL)

DDL (Data Definition Language) se utiliza para definir la estructura de la base de datos (tablas, esquemas, índices, etc.).

CREATE

Crea nuevos objetos en la base de datos, como tablas. 


CREATE TABLE users (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE
);

ALTER

Modifica la estructura de un objeto existente. 


ALTER TABLE users ADD COLUMN created_at DATE;

DROP

Elimina un objeto de la base de datos permanentemente. 


DROP TABLE old_data;

TRUNCATE

Elimina todos los registros de una tabla, pero mantiene su estructura. Es más rápido que DELETE. 


TRUNCATE TABLE logs;

COMMENT

Añade comentarios al diccionario de datos. 


COMMENT ON COLUMN users.email IS 'Email address of the user';

RENAME

Cambia el nombre de un objeto. 


RENAME TABLE users TO customers;
