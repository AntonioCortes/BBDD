Data Query Language (DQL)

DQL (Data Query Language) son instrucciones SQL utilizadas para obtener datos de la base de datos. El comando principal es SELECT.

SELECT

Se utiliza para recuperar registros de una o más tablas. 


SELECT column1, column2 FROM table_name;
Ejemplo de consulta básica

También podemos filtrar resultados con WHERE: 


SELECT * FROM employees WHERE department = 'IT';
