Data Manipulation Language (DML)

DML (Data Manipulation Language) se utiliza para manipular los datos dentro de las tablas. Los comandos principales son INSERT, UPDATE y DELETE.

INSERT

Inserta nuevos registros en una tabla. 


INSERT INTO users (name, email) VALUES ('Juan', 'juan@example.com');

UPDATE

Modifica registros existentes en una tabla. 


UPDATE products SET price = 20 WHERE id = 1;

DELETE

Elimina registros de una tabla. 


DELETE FROM orders WHERE status = 'cancelled';
